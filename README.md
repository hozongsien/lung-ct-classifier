# Quick Start

1. Setup environment.
   ```
   pip install -r requirements.txt
   ```
2. Download data and unzip files.
   ```
   unzip -a nus-cs5242.zip -d data/raw
   ```
3. Preprocess images and create dataset.
   ```
   bash prepare_dataset.sh
   ```
4. Train model using jupyter notebook.
   ```
   jupyter lab
   ```
